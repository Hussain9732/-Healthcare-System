🏥 Smart Healthcare System (OOP Project)
📌 Overview

The Smart Healthcare System is a console-based application developed using Object-Oriented Programming (OOP) in Java.
It simulates a basic healthcare environment where patients can book appointments with doctors, and prescriptions are generated based on symptoms.

⚙️ Features
👤 Patient and Doctor management system
🏥 Multiple doctor types (Cardiologist, General Physician)
📅 Online and Physical appointment system
💊 Prescription generation based on symptoms
❗ Custom exception handling for invalid operations
💾 File handling for storing appointment and prescription records
🔒 Input validation for secure data handling
🧠 Concepts Used
Object-Oriented Programming (OOP)
Inheritance
Abstraction
Polymorphism
Encapsulation
Exception Handling (Custom Exceptions)
File Handling (Java I/O)
Class Design & System Modeling
